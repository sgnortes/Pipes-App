Pipes Básicos

Pipes incluídos en Angular - usualmente en el Common Module.

A continuación mostraremos algunos ejemplos de valores de entrada y salida en función del pipe escogido:

pipe usado: nombre | uppercase
ejemplo resultado: {{ nombreLower | uppercase }}
pipe usado: nombre | lowercase
ejemplo resultado: {{ nombreUpper | lowercase }}
pipe usado: nombre | titlecase
ejemplo resultado: {{ nombreCompleto | titlecase }}

Date pipe

Ejemplos de pipes para formatear las fechas.

  • fecha
  • fecha | date
  • fecha | date:'short'
  • fecha | date:'long'
  • fecha | date: 'yyyy/mm/dd'
  • fecha | date:'long':'GMT-6'
  • fecha | date: 'long':'':'fr'
  • {{ fecha }}
  • {{ fecha | date }}
  • {{ fecha | date: "short" }}
  • {{ fecha | date: "long" }}
  • {{ fecha | date: "yyyy/mm/dd" }}
  • {{ fecha | date: "long":"GMT-6" }}
  • {{ fecha | date: "long":"":"fr" }}
