Pipes no tan comunes

Pipes incluídos en Angular - usualmente en el common module

Vamos a cambiar género de verbo invitar en función del nombre:

Saludos {{ nombre }}, es un placer {{ genero | i18nSelect: invitacionMapa }} a nuestro evento.

En función de la cantidad de clientes escribiremos la oración en singular o plural:

Actualmente {{ clientes.length | i18nPlural: clientesMapa }}.
Original 
{{ clientes }}
Slice: 0:2 
{{ clientes | slice: 0:2 }}

Lo usamos para imprimir las distintas propiedades de un objeto.

  • {{ item.key | titlecase }}: {{ item.value }}

Lo usamos para pintar objetos en formato json.

{{ heroes | json }}

Lo usamos para suscribirnos a observables y mostrar lo que emiten. 

 {{ miObservable | async }} 
Resolviendo la promesa
{{ valorPromesa | async }}
